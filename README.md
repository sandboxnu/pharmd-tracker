# pharmd-tracker

**To run the front end:**

**In pharmd-app, run:**

- npm install
- npm start

**Afterward**

- Install CORS plugin
- link to [chrome plugin](https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf?hl=en)
- Turn it on by clicking on the C icon
